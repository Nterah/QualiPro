Help

Quick guide to the main areas.

  • Project Selector: Search and open a project’s PQP by code.
  • Import Center: Upload one or many PQP workbooks, preview mappings, then commit to the database.
  • PQP Form: View/Edit a single project’s nine sections (add rows, import per section, export).
  • Export Center: Download a ZIP of CSVs or an HTML summary for a project.
  • Reminders: Send overdue emails (demo endpoint out-of-the-box).
  • Settings: SMTP and export defaults.

Tip: you can print the HTML summary and “Save as PDF”.
